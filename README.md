# Lexor

Legal-adjacent counsel build with **`lexor-web`**, **`lexor-demo-video`**, `api-server`, and `mockup-sandbox`—counsel UX and media artifacts on the shared stack.

## 💼 Real-World Use Case
Teams exploring counsel workflows or explainer videos tied to structured case/intake data models (demo-grade; not a substitute for licensed practice software).

## 💰 Potential Value
- Vertical SaaS potential paired with Briefcased-style evidence flows—**only** with compliance, retention, and jurisdiction review.
- Shows multimedia + web packaging in one monorepo.

## 🚀 Demo / Access
Self-hosted / local only. Configure `artifacts/api-server/.env` and follow pnpm dev flows. No hosted demo URL is pinned.
